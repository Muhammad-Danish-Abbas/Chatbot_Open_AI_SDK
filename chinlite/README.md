## 🚀 Quick Setup Guide

Follow these steps to set up your chatbot:

### 1. Initialize the Project
```bash
uv init --package chatbot
cd chatbot
code .
```
### 2. Run Initial Test
```bash
uv run chatbot
```
###3. Install Required Packages
Run in PowerShell:
```bash
uv add openai-agents
uv add chainlit

```
###4. Test Chainlit Installation
```bash
uv run chainlit hello

```
